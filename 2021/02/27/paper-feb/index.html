Deep Residual Learning for Image Recognition

Problem

  • Deep networks are more difficult to train [vanishing/exploding gradients problem], rethinking deep representation is really important ?

  • degradation problem definition

    • With the network depth increasing, accuracy gets saturated (which might be unsurprising) and then degrades rapidly. Unexpectedly,degradation is not caused by overfitting.

  • The degradation problem suggests that the solvers might have difficulties in approximating identity mappings by multiple nonlinear layers.

So in this paper,the most contribution is deep residual learning framework. ** ,which are easier to optimize, and can gain accuracy from considerably increased depth.With the residual learning reformulation, if **identity mappings are optimal, the solvers may simply drive the weights of the multiple nonlinear layers toward zero to approach identity mappings.

Motivation

  • residual representation
    • VLDA and Fisher Vector Both of them are powerful shallow representations for image re- trieval and classification .VLAD **is a representation that encodes by the residual vectors with respect to a dictionary ；Fisher Vector ** can be formulated as a probabilistic version [18] of VLAD.
    • Vector Quantization, encoding residual vectors [17] is shown to be more effec- tive than encoding original vectors.
    • In PDE, Multigrid Method reformulates the system as subprob- lems at multiple scales, where each subproblem is respon- sible for the residual solution between a coarser and a finer scale.

These methods suggest that a good reformulation or preconditioning can simplify the optimization.

  • Shortcut connections:

    • In fact ,residual connection is not totally a novel idea.An early practice of training MLP is to add a linear layer connected from the network input to the output; Inception Network,layer is com- posed of a shortcut branch and a few deeper branches. ; Highway Network present shortcut connections with gating functions ;

    Identity shortcuts that are parameter-free,and never closed. all information is always passed through, with additional residual functions to be learned.

AN IMAGE IS WORTH 16X16 WORDS: TRANSFORMERS FOR IMAGE RECOGNITION AT SCALE

Problem

Transfomrer has already become a deco-facto standard in NLP, it’s application in cv is limited. 

self-attention , Transformers’  computational efficiency and scalability,

Pure transformer[[not combined with CNNs] applied directly to sequences of image patches can perform very well on image classification tasks ?

Existing approches and Challenges

  • self-attention to images would require that each pixel attends to every other pixel. With quadratic cost in the number of pixels, this does not scale to realistic input sizes
  • combining convolutional neural networks (CNNs) with forms of self-attention

Motivation

Method

  • PipeLine
    • Step1:split an image into fixed-size patches, linearly embed each of them, add position embeddings
      • reshape image 3d ->2d: $x\in R^{H \times W \times C} -> R^{(HW/p^2) \times (p^2C)}$ which $N = \frac{HW}{p^2}$ is the number of the patches
      • Learnable Position Eembedding : 2d position vs 3d position ?
    • Step2: feed the resulting sequence of vectors to a standard Transformer encoder
    • Finally : adding an extra learnable “classification token” to the sequence.

  • Pretrained in large Datasets
    • ImageNet, ImageNet-21k, CIFAR100,JFT- 300M
  • Results
    • Fine-Tuning: optimize three regularization parameters – weight decay, dropout, and label smoothing
    • Replace Batch Normalization Layers in ResNet with Group Normalization will imporve ResNet performance.

Question & Conclusion

  • First Transformers with global self-attention to full-sized images

  • ViT approaches or beats state of the art on multiple image recognition benchmarks. In particular, the best model reaches the accuracy of 88.55% on ImageNet, 90.72% on ImageNet-ReaL, 94.55% on CIFAR-100, and 77.63% on the VTAB suite of 19 tasks.
